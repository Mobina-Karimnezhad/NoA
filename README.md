# NoA

NoA is an AI-powered habit tracker application developed as a Bachelor's Final Year Project.

## Features

- Habit Tracking
- Streak System
- Smart AI Assistant
- Statistics
- Daily Reminders
- Persian UI
- Material 3 Design

## Tech Stack

- Kotlin
- Jetpack Compose
- MVVM
- Clean Architecture
- Room
- Hilt
- DataStore
- Retrofit
- OpenRouter API
